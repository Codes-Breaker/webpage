DEADLY SHOTS

Alien Shooting Game

Harry Li

Overview

This is the first original game I built using XCode when I was in high school. My initial idea was to build a tapping game to simulate shooting on my iPhone. This game would generate random numbers of aliens on the screen and disappear if the player fails to kill it within few seconds. After 30 seconds, the game would stop and return player's score. I also implemented multiplayer mode after I finished this project using python as the server.
